Atributo Box-Shadow

Sombras em Caixas Utilizando CSS3

As sombras são muito úteis para dar volume, para dar a sensação de que as caixas estão ali realmente.

Para criar uma sombra numa caixa, vamos utilizar uma declaração especial, box-shadow.

Assim que adicionamos a propriedade box-shadow e seus quatro valores, a ordem é sempre essa:

1- Deslocamento horizontal (h-offset): quando a sombra vai andar para o lado direito (valores negativos causam deslocamento para a ESQUERDA)

2- Deslocamento vertical (v-offset): quando a sombra vai andar para baixo (valores negativos causam deslocamento para CIMA)

3- Embaçamento (blur): quando a sombra se espalha pelo fundo.

4- Cor (color): cor da sombra. É possível usar transparência.
